** I customized my keyboard, so it might be slightly different from yours.

Default layer

Fn + Esc = `

Fn + 1 = F1

Fn + 2 = F2

Fn + 3 = F3

Fn + 4 = F4

Fn + 5 = F5

Fn + 6 = F6

Fn + 7 = F7

Fn + 8 = F8

Fn + 9 = F9

Fn + 0 = F10

Fn + - = F11

Fn + = = F12

Fn + BACKSPACE = DEL

Fn + W = UP

Fn + S = LEFT

Fn + A = DOWN

Fn + D = RIGHT

Fn + P = Print Screen

Fn + [ = Scroll Lock

Fn + ] = Pause

Fn + ; = Home

Fn + . = End

Fn + ' = Page Up

Fn + / = Page Down

Fn + N = Volume Down

Fn + M = Volume Up

Fn + , = Mute

Fn + Y = Calculator

Fn + X = APP

Fn + I = Ins # Don’t know what it is for

Fn + B = Increase LED brightness

Fn + V = Reduce LED brightness

Fn + C = Change LED mode

Layer 1 (Windows)

** Haven’t finish modification yet.

CAPS = Fn

Fn = Pn

Pn = CAPS

Layer 2 (Mac)

** Haven’t finish modification yet.

CAPS = Fn

Fn = Pn

Pn = CAPS

Layer 3

** Haven’t finish modification yet.

CAPS = Fn

Fn = Pn

Pn = CAPS

Function Keys

Fn + R = Restore current layer to factory default # Need to hold for 5 sec

Left Alt + Right Alt = Restore all layers to factory default # Need to hold for 5 sec

Fn + Right SHIFT = Switch Layer

Fn + Right CTRL = Enter/Exit Programming Mode

Fn + F = Delay 15ms

Fn + G = Delay 0.1s

Fn + H = Delay 0.5s

Reprogramming

Fn + Right CTRL –> ( Target Key –> Key Content –> Pn to save ) * repeat –> Fn + Right CTRL

Switch

Change the keyboard layout.

Switch 1 Switch 2 Keyboard Layout
ON ON Workman
ON OFF Dvorak
OFF ON Colemak
OFF OFF Qwerty

Change Pn and Fn

Switch 3 ON –> Pn / Fn –> Target Key –> Switch 3 OFF

Enable arrow key

When Switch 4 is ON: # Default layer

Right Alt = Fn

Right SHIFT = UP

Fn = LEFT

Pn = DOWN

Right CTRL = RIGHT

Swap Left CTRL and CAPS

When Switch 5 is ON: # Default layer

CAPS = CTRL

Left CTRL = CAPS

Swap Win and Alt

When Switch 6 is ON: # Default layer

Left Alt = Win

Left CTRL = Alt

Right Alt = Win
